# Personal Portfolio Website

A responsive and interactive personal portfolio website built to showcase my skills, projects, and experience as a Frontend Developer.
The website focuses on clean UI, smooth interactions, and user-centric design.

# Features

Professional landing section with personal introduction

About section highlighting skills & background

Projects section with real-world projects

Interactive UI elements & animations

Fully responsive (mobile, tablet & desktop)

Contact form for easy communication

Fast loading & optimized performance

# Tech Stack

HTML5

CSS3

JavaScript

Bootstrap

Firebase Hosting

FormSubmit API (for contact form handling)

# How It Works

Portfolio loads with a responsive landing section

Users can navigate through sections smoothly

Projects are displayed with clear descriptions

Contact form submits messages using FormSubmit API

Website is hosted and served via Firebase

# Learning Outcomes

Building a complete personal brand website

Implementing responsive layouts

Hosting & deployment using Firebase

Creating user-centric UI designs
